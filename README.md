# Student Portfolio (Node.js Backend)

This is a Node.js backend project for managing student portfolio data.

## Features
- Node.js backend
- Express server
- Portfolio data management
- REST API support

## Project Structure

server/
public/
views/
package.json

## Installation

1. Clone the repository
2. Install dependencies

npm install

3. Start the server

npm start
